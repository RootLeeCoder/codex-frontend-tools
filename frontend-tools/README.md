# 前端工具合集

基于 [Vue 3](https://vuejs.org/)、[Element Plus](https://element-plus.org/) 和 [TypeScript](https://www.typescriptlang.org/) 构建的前端在线工具导航项目。

## 开始使用

```bash
npm install
npm run dev
```

## 脚本命令

- `npm run dev`：启动本地开发服务器。
- `npm run build`：构建生产环境静态资源。
- `npm run preview`：预览构建后的站点。
- `npm run lint`：执行 ESLint 代码检查。
